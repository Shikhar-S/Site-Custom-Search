Search API

We also provide a search API, which returns JSON response corresponding to your search queries. Below is a short example on how to use this service. Experienced web developers can use this service if they want complete control over the design of the search landing page.

import requests search_url = "http://127.0.0.1:8000/search/" search_term = "hello" import requests params = {"search_term": search_term,"crawler_id":"Your Crawler ID" ,"numres":5 } response = requests.post(search_url,data=params) # response.raise_for_status() search_results = response.json() print(search_results).

Just Search Bar with popup results
